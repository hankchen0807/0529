學習過程分享

網頁設計的過程充滿挑戰與樂趣。從學習HTML的結構開始，到深入CSS的排版與動畫效果，每一步都讓我感受到創作的魅力。

我透過網路資源、自學影片與實際操作，不斷調整與測試，終於完成了多個作品，並能夠靈活應用各種技巧。

課程重點筆記

  • HTML 結構：包含 head、body、header、footer、section 等基本標籤。
  • CSS 選擇器：ID (#)、類別 (.)、元素選擇器、群組與巢狀選擇器。
  • 排版技巧：使用 flexbox 或 grid 進行版面配置，掌握 float 和 clear 的運用。
  • RWD 響應式設計：利用 @media 查詢讓網頁在不同裝置上自動調整。
  • 網頁美化：使用 box-shadow、border-radius、背景漸層、hover 效果增強視覺吸引力。
